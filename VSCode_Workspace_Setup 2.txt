VS Code & GitHub Copilot Instant Setup Guide
This document contains the instructions and automated script to instantly configure your workspace with the 21st.dev Magic and Stitch MCP servers, along with the GSAP & Framer Motion Scroll Animation Copilot skill.
One-Click Install Script (Terminal)
Open your terminal, navigate to your project's root folder, and paste the following commands to instantly install everything:
# 1. Setup Custom Copilot Skills Directory
mkdir -p .github/skills/gsap-framer-scroll-animation

# Download the GSAP Framer Motion Skill
curl -sL https://raw.githubusercontent.com/github/awesome-copilot/main/skills/gsap-framer-scroll-animation/SKILL.md -o .github/skills/gsap-framer-scroll-animation/SKILL.md

# 2. Setup MCP Servers Configuration
mkdir -p .vscode
cat << 'MCPJSON' > .vscode/mcp.json
{
  "mcpServers": {
    "@21st-dev/magic": {
      "command": "npx",
      "args": [
        "-y",
        "@21st-dev/magic@latest",
        "0c44df82feeafd6daae58057f48e4a652638054e0847f8b6d5e7158010afc1de"
      ]
    },
    "stitch-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "stitch-mcp"
      ],
      "env": {
        "STITCH_API_KEY": "REPLACE_WITH_YOUR_STITCH_API_KEY"
      }
    },
    "TestSprite": {
      "command": "npx",
      "args": [
        "@testsprite/testsprite-mcp@latest"
      ],
      "env": {
        "API_KEY": "REPLACE_WITH_YOUR_TESTSPRITE_API_KEY"
      }
    }
  }
}
MCPJSON

echo "✅ Setup Complete! Please run 'Developer: Reload Window' in VS Code to refresh MCP servers."
Manual Verification Steps
	1	Verify that the .vscode/mcp.json file exists and contains the valid JSON configuration.
	2	Check that the skill exists at .github/skills/gsap-framer-scroll-animation/SKILL.md.
	3	In VS Code, open the Command Palette (Cmd+Shift+P) and select Developer: Reload Window to ensure Copilot registers the newly added MCP servers.
You can now use Copilot chat prompts like: "Create a modern dashboard sidebar using shadcn and tailwind" and it will auto-trigger the connected skills and servers.
